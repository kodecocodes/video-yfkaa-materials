# Your First Kotlin Android App: An App From Scratch: Materials

This repo contains all the downloadable materials and projects associated with the **[Your First Kotlin Android App: An App From Scratch](https://www.raywenderlich.com/30498263-your-first-kotlin-android-app-an-app-from-scratch)** from [raywenderlich.com](https://www.raywenderlich.com).

Each edition has its own branch, named `versions/[VERSION]`. The default branch for this repo is for the most recent edition.
